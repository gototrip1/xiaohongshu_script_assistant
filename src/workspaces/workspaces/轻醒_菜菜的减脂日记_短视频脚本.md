## 二、完整口播文案

### 【0-5s】开场钩子

> **口播**：「姐妹们！减脂早餐第28天！今天这个搭配我真的要吃一辈子！」
>
> **语气**：兴奋、日常，像跟闺蜜分享新发现

### 【5-15s】场景痛点 + 便当展示

> **口播**：「打工人早上真的起不来对吧？今天这个5分钟就能搞定，高蛋白低负担，饱腹感绝了，一整个上午都不会饿。」
>
> **语气**：共鸣感、实用导向

### 【15-30s】制作过程 + 产品第一次露出

> **口播**：「先准备一盒轻醒的希腊酸奶，0蔗糖的哈，这个蛋白质含量很高。倒到碗里，你看这个质地——」
>
> **语气**：自然、随性，像在展示自己日常在用的东西

### 【30-45s】搭配过程 + 营养数据

> **口播**：「加点蓝莓和燕麦片进去，这个蓝莓味的轻醒本身就带果粒，但我还是喜欢自己再加一点。一杯下来蛋白质大概有15g左右，碳水也控制住了。」
>
> **语气**：轻松科普、数据化表达

### 【45-55s】成品展示 + 产品第二次露出

> **口播**：「你看，就这么简单。旁边再放一盒黄桃味的当加餐，下午饿的时候来一盒，比零食健康多了。」
>
> **语气**：满足感、分享感

### 【55-65s】食用镜头 + 真实感受

> **口播**：「我先替你们尝一口——嗯，浓稠度真的很可以，不是那种稀的酸奶，吃起来像希腊酸奶该有的样子。」
>
> **语气**：真实体验、自然种草

### 【65-75s】互动引导 + 收尾

> **口播**：「姐妹们你们早餐一般都怎么搭配？评论区交流一下！咱们一起打卡，明天见！」
>
> **语气**：亲切、邀请感

## 六、风险质检

### 6.1 合规风险检查

- [x] 无绝对化用语 — 未使用"最好""第一""绝对"等词
- [x] 无虚假承诺或夸大宣传 — 未承诺减重、降糖等功效
- [x] 无医疗/健康领域的违规功效宣称 — 仅描述"高蛋白""0蔗糖"等客观属性
- [x] 产品功效描述有依据支撑 — "蛋白质≈15g"基于希腊酸奶品类通用数据
- [x] 无竞品贬低 — 未提及其他品牌

### 6.2 内容质量检查

- [x] 符合博主"菜菜的减脂日记"风格 — 不露脸、俯拍、连续剧打卡式、口语化
- [x] 产品植入自然 — 酸奶作为早餐食材之一出现，非单独硬推
- [x] 多口味展示 — 蓝莓味（主镜头）+ 黄桃味（加餐场景），自然不刻意
- [x] 拍摄门槛低 — 仅需手机+俯拍支架，无需专业设备
- [x] 时长合规 — 约75秒，在60-90秒范围内
- [x] 互动引导到位 — 结尾设置开放式问题引导评论
